# Системный анализ факторов успеха IT-стартапов в Казахстане

Данный репозиторий содержит материалы научно-исследовательского проекта, посвященного анализу выживаемости и ключевых факторов успеха технологических компаний.

## 📁 Структура проекта
* **data/** — исходные данные для анализа в формате CSV (startups_kz.csv).
* **notebooks/** — программная реализация модели логистической регрессии на языке Python.
* **docs/** — пояснительная записка, презентация и финальный отчет в формате PDF.

## 🛠 Технологический стек
* **Язык программирования:** Python 3.10
* **Библиотеки:** Pandas (обработка данных), Scikit-learn (машинное обучение), Matplotlib (визуализация).
* **Оформление отчета:** LaTeX (Overleaf)[cite: 1].
